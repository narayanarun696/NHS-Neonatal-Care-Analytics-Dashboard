# NHS-Neonatal-Care-Analytics-Dashboard
An interactive Power BI dashboard analysing neonatal admissions and clinical indicators across NHS Trusts in South West England. All data has been fully anonymised.
This dashboard was built to explore patterns in neonatal care across NHS Trusts in the South West of England. It provides a visual summary of admission types, length of stay, and clinical reasons for admission — helping to identify trends and support data-driven decision-making in neonatal care.

# Breakdown of admissions across each NHS Trust by category:
TermAds – Term admissions
NNU Short Stay – Neonatal Unit short stay admissions
NNU Long Stay – Neonatal Unit long stay admissions

# Visualisations Include
